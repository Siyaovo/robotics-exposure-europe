# Data

Raw data files are not tracked in this repository due to file size.  
Place all downloaded files in `data/raw/` before running the notebooks.

---

## Download Instructions

### 1. ESCO — European Skills, Competences, Qualifications and Occupations
- **Source:** https://esco.ec.europa.eu/en/use-esco/download
- **Version:** ESCO dataset v1.2.1
- **Content:** Classification
- **File type:** CSV · **Language:** EN
- **Files needed:**
  - `occupations_en.csv`
  - `skills_en.csv`
  - `occupationSkillRelations_en.csv`

### 2. O*NET Task Statements
- **Source:** https://www.onetcenter.org/database.html
- **Section:** Download → Task Statements
- **File needed:** `Task Statements.xlsx`

### 3. Eurostat — Mean Hourly Earnings by Occupation (SES 2022)
- **Source:** https://ec.europa.eu/eurostat/databrowser
- **Path:** Population and social conditions → Labour market → Earnings → Structure of earnings survey 2022 → Hourly earnings → `earn_ses22_14`
- **Filters:** Sex = Total · Age = Total · Size class = GE10 · Year = 2022
- **File needed:** `estat_earn_ses22_14.tsv`
