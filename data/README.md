# Data

Raw data files are not tracked in this repository due to file size.

## Download instructions

### ESCO (European Skills, Competences, Qualifications and Occupations)
- Source: https://esco.ec.europa.eu/en/use-esco/download
- Version: ESCO dataset v1.2.1
- Files needed:
  - `occupations_en.csv`
  - `skills_en.csv`
  - `occupationSkillRelations_en.csv`

### O*NET Task Statements
- Source: https://www.onetcenter.org/database.html
- File needed: `Task Statements.xlsx`

### Eurostat
- Wages by occupation: https://ec.europa.eu/eurostat/databrowser/view/earn_ses_pub1d
- Employment by occupation: https://ec.europa.eu/eurostat/databrowser/view/lfsa_egais

## Folder structure
Place all downloaded files in `data/raw/` before running the notebooks.
